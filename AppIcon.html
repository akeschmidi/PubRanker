📱 So verwenden Sie das Icon:

  1. Wählen Sie Ihr favorisiertes Icon-Design
  2. Rechtsklick auf das SVG → "Als Bild speichern"
  3. Öffnen Sie die Datei in Vorschau oder Pixelmator
  4. Exportieren Sie als PNG in 1024x1024 Pixel
  5. In Xcode: Assets.xcassets → AppIcon → Drag & Drop das 1024x1024 PNG
  6. Xcode generiert automatisch alle benötigten Größen

💡 Alternative: icongen.app

Oder nutzen Sie ein Online-Tool:

  1. Speichern Sie das SVG
  2. Gehen Sie zu https://www.appicon.co/
  3. Laden Sie das 1024x1024 PNG hoch
  4. Download macOS Icon Set
  5. Drag & Drop in Xcode

🎨 Empfehlung:

Ich empfehle Option 1 (Trophy) oder Option 4 (Ranking) - beide kommunizieren klar die App-Funktion und sehen auf dem Mac Dock professionell aus.
